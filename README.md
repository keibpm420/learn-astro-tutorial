# Astro 学習ブログ 🚀

Astro 公式チュートリアルを通じて構築した学習用ブログプロジェクトです。

デモページ：[https://learn-astro-tutorial.vercel.app/](https://learn-astro-tutorial.vercel.app/)

## 🛠 使用技術

-   **Framework:** [Astro](https://astro.build/)
-   **Styling:** CSS (Astro 固有のスタイル、グローバルスタイル)
-   **Deployment:** [Vercel](https://vercel.com/)
-   **Features:**
    -   RSS フィード生成 (`@astrojs/rss`)
    -   ダークモード切り替え機能
    -   動的ルーティングによる記事生成
    -   コンポーネントベースの設計 (Header, Footer, Navigation 等)

## 🚀 ローカルでの開発
